Bootstrap tutorial 7 - Push and Pull

This is really useful for mobile optimisation, with articles/images. Using push and pull to swap column positions, and having a medium breakpoint for the columns, for example, means that an article and an image, for example, will appear adjacent but when the screen is below medium size, the columns will swap back AND stack, with the image on top.

This example uses the css class 'bg-cover' to pull the image, and the 'col-md-push-8' IN the div. However, as the image height is restricted to 300px, part of it is cut off when the screen size is reduced.

Column 1 - the image

Column 2 - An Article for example

This example uses the html img element to pull the image, inside the 'col-md-push-8' div. This keeps the whole image visible when the screen size is reduced.

Column 2 - An Article for example

With the article and image swapped position in the html, and without the PUSH class in the image div and the PULL class in the article, when the screen size is reduced the article then appears above the image instead of below as intended. and the text doesn't start till after the 8 col size
